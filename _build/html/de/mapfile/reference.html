REFERENCE¶

Es werden drei Typen von Referenzkarten unterstützt. Die am häufigsten benutzte Variante zeigt den Extent der Karte in einer interaktiven Anwendung. Darüber hinaus kann auch die Übersichtskarte mit einer Abfrage (Query) vom MapServer angefordert werden. Punktabfragen erzeugen ein Bild mit einem Marker (siehe unten) an der Abfragekoordinate. Flächenbasierte Abfragen erzeugen ein Bild das den Extent der abgefragten Fläche darstellt. Objektabfragen stellen die abgefragten Objekte dar.

COLOR [r] [g] [b]
Farbe mit der die Referenzbox gezeichnet wird. Ein Farbwert von -1 bedeutet keine Füllung. Standard ist rot.
EXTENT [minx][miny][maxx][maxy]
Die räumliche Ausdehnung des Basisreferenzbildes.
IMAGE [filename]
Vollständiger Dateiname des Basisreferenzbildes. Sollte im GIF Format vorliegen.
MARKER [integer|string]
Definiert ein Symbol (aus der Symboldatei) das benutzt wird wenn die Box zu klein wird (siehe MINBOXSIZE und MAXBOXSIZE). Als Standard wird ein Fadenkreuz benutzt.
MARKERSIZE [integer]
Definiert die Größe des Symbols das anstatt der Box benutzt wird (siehe MARKER).
MINBOXSIZE [integer]
Ist die Box kleiner als durch MINBOXSIZE (Boxbreite oder Boxhöhe) definiert, wird das durch MARKER und MARKERSIZE definierte Symbol benutzt.
MAXBOXSIZE [integer]
Ist die Box größer als durch MAXBOXSIZE (Boxbreite oder Boxhöhe) definiert, wird nichts gezeichnet. (Oftmals wird die gesamte Karte angezeigt, wenn man weit genug herauszoomt. Dadurch erschließt sich sofort wo man sich befindet).
OUTLINECOLOR [r] [g] [b]
Farbe mit der der Rahmen der Referenzbox gezeichnet wird. Ein Farbwert von -1 bedeutet kein Rahmen.
SIZE [x][y]
Größe des Basisrefenrenzbildes in Pixel.
STATUS [on|off]
Gibt an ob die Referenzkarte erstellt wird. Standard ist aus.
