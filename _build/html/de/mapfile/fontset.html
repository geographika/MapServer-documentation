Format¶

Das Format ist recht einfach. Jede Zeile enthält zwei Einträge: einen Aliasnamen und den Namen bzw. Pfad der Schriftdatei, getrennt durch Leerzeichen. Der Aliasname wird im Mapfile als Referenz benutzt. Der Name ist der tatsächliche Name der TrueType Datei. Wird kein absoluter Pfad angegeben, wird die Angabe relativ zum FONTSET ausgewertet. Bemerkung

Aliasnamen sind abhängig von Groß-/Kleinschreibung. Sehr ausführliche Informationen zum TrueType Format sind unter FreeType abrufbar.
