sortshp¶

Zweck:

Sortiert ein Shapefile nach einer Spalte auf- oder absteigend. Es werden INTEGER, DOUBLE und STRING Spaltentypen unterstützt. Nützlich falls bestimmte Shapes zum rendern oder beschriften priorisiert werden sollen.
Beschreibung:

Wird ein bestimmter Wert einer bestimmten Attributklasse zum Beschriften benötigt, so kann MapServer schneller auf diesen Wert zugreiffen, wenn er am Anfang der Attributtabelle steht.
Syntax:
sortshp [infile] [outfile] [item] [ascending|descending]
Beispiel:

Dieses Beispiel benutzt eine Straßendatei (‘roads_ugl’) mit einem Feld für Strassenklassen im Integerformat (‘class1’).

sortshp roads_ugl roads-sort class1 ascending
Ergebnis:

Ein neues Shapefile ‘roads-sort.shp’ wird erstellt. Alle Elemente sind in aufsteigender Reihenfolge sortiert, in Bezug auf die Spalte ‘class1’,
