Bienvenidos a Mapserver¶

MapServer es una plataforma de Código Abierto para la publicación de datos espaciales y aplicaciones cartográficas interactivas para la web. Originalmente desarrollado a mediados de los 90’s en la Universidad de Minnesota, MapServer es publicado bajo una licencia tipo MIT, y funciona en los principales sistemas operativos (windows, linux, osx).

El Comité Directivo del Proyecto (PSC) MapServer, avalado por OSGeo, gestiona y administra el proyecto, el cual es mantenido, mejorado y apoyado por desarrolladores de todo el mundo. MapServer no es un sistema con todas las funcionalidades de un sistema SIG, ni tampoco aspira a serlo.
